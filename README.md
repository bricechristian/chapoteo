# Chapoteo (Splash)

A small spanish vocabulary flash card app to help me memorize the most popular 1000 spanish words.

It uses:
- [Next JS](https://nextjs.org/)
- SCSS + [Tailwind](https://tailwindcss.com/)
- [Merriam-Webster API](https://dictionaryapi.com/)
